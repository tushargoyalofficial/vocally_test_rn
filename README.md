# Vocally Test

## Tech used
1. Expo sdk 45
2. react-native-maps
3. react-navigation v6

## Development time
Approximatly 3 Hrs

## Command to run project
1. take pull of the project
2. run "yarn install" from root project dir
3. run "expo start" from root project dir
4. Open "Expo Go app" on your device
5. Scan the QR code
